# Key Features of a Geometry Toolkit

Educational MicroSims offer an alternative to many expensige 

1. **Ease of use** - You should be able to add a MicroSim to ANY webpage with just a singe HTML `iframe` statement.
2. **Permissive License** - You should be able to use any and all MicroSims to your class without fees
3. **Customization** - You should be able to customize the MicroSim to meet your needs
