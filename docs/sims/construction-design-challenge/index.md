## About This MicroSim

This project-based challenge applies geometric construction techniques to design a symmetric bridge structure. Use bisection, perpendicular construction, and parallel lines to create a structurally sound design.

## The Four Phases

| Phase | Objective | Construction Used |
|-------|-----------|-------------------|
| **1** | Find center C | Bisect segment AB |
| **2** | Build tower | Perpendicular at C |
| **3** | Add braces | Symmetric lines AT, BT |
| **4** | Complete | Parallel horizontal beams |

## Key Constructions Applied

- **Segment bisection**: Finding the midpoint ensures symmetry
- **Perpendicular construction**: Vertical tower at 90Â° to ground
- **Angle copying**: Braces make equal angles with ground
- **Parallel lines**: Horizontal beams parallel to ground
