## About This Infographic

This interactive infographic explores how angle relationships appear in six different real-world contexts. Click on any category to see detailed examples, fun facts, and visual demonstrations.

## Categories

| Category | Examples | Key Angle Types |
|----------|----------|-----------------|
| **Architecture** | Building corners, roof pitches, arches | Right angles, Acute angles |
| **Navigation** | Compass bearings, course changes, coordinates | Full rotation, Bearing angles |
| **Art & Design** | Perspective, symmetry, golden angle | Acute angles, Supplementary |
| **Engineering** | Gears, bridges, robotics | Complementary, Acute angles |
| **Sports** | Shooting angles, trajectories, racing | Acute angles, Obtuse angles |
| **Daily Life** | Clock hands, ramps, viewing angles | Right angles, Acute angles |
