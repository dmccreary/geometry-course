## About This MicroSim

This interactive tool explores the four types of angle relationships formed when a transversal crosses two parallel lines. Click buttons to highlight different angle pairs and adjust the transversal angle with the slider.

## Angle Relationships

| Relationship | Pairs | Property |
|--------------|-------|----------|
| **Corresponding** | ∠1&∠5, ∠2&∠6, ∠3&∠7, ∠4&∠8 | Congruent (≅) |
| **Alt. Interior** | ∠3&∠6, ∠4&∠5 | Congruent (≅) |
| **Alt. Exterior** | ∠1&∠8, ∠2&∠7 | Congruent (≅) |
| **Same-Side Interior** | ∠3&∠5, ∠4&∠6 | Supplementary (180°) |

## How to Use

1. **Click buttons** to highlight different angle relationships
2. **Adjust the slider** to change the transversal angle
3. **Observe** that relationships hold for any transversal angle
4. **Click Clear** to remove highlighting

## Key Insight

When parallel lines are cut by a transversal:
- Three pairs are **congruent** (equal)
- One pair is **supplementary** (adds to 180°)

## Learning Objectives

- **Identify** the four angle relationship types
- **Apply** angle relationships to find unknown measures
- **Verify** that relationships hold for any transversal angle

## Bloom's Taxonomy Level
