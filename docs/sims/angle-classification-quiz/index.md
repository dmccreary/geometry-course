## About This MicroSim

This interactive quiz tests your ability to classify angles based on their measure. Each question shows a random angle, and you must identify whether it's acute, right, obtuse, or straight.

## Angle Types

| Type | Measure Range | Color |
|------|---------------|-------|
| Acute | 0° < x < 90° | Green |
| Right | x = 90° | Blue |
| Obtuse | 90° < x < 180° | Orange |
| Straight | x = 180° | Purple |

## How to Play

1. **Look** at the angle displayed and its measurement
2. **Click** the button matching the correct classification
3. **Read** the feedback to understand why
4. **Click** "Next Angle" for a new question
5. **Track** your score in the top right

## Interactive Features

- **Randomized angles** - Different angles each time
- **Immediate feedback** - Learn from correct and incorrect answers
- **Score tracking** - See your progress
- **Visual representation** - See the angle as you classify

## Learning Objectives

- **Classify** angles by their measure
- **Distinguish** between acute, right, obtuse, and straight angles
- **Apply** angle classification rules quickly

## Bloom's Taxonomy Level

**Understanding** and **Applying** - Classifying angles based on learned definitions.
