## About This Concept Map

This interactive concept map visualizes how all 14 angle concepts from Chapter 03 relate to each other. The hierarchical layout shows dependencies between concepts, helping you understand prerequisite relationships.

## Concept Categories

| Category | Shape | Color | Concepts |
|----------|-------|-------|----------|
| **Base** | Ellipse | Purple | Angle |
| **Classifications** | Square | Blue | Acute, Right, Obtuse, Straight |
| **Relationships** | Ellipse | Green | Complementary, Supplementary, Adjacent, Vertical, Linear Pair |
| **Advanced** | Diamond | Orange | Bisector, Corresponding, Alt. Interior, Alt. Exterior, Same-Side |

## Edge Types

| Edge Type | Meaning | Example |
|-----------|---------|---------|
| **is-a** | Classification relationship | "Acute Angle is-a Angle" |
| **uses** | One concept uses another | "Complementary uses Acute" |
| **requires** | Prerequisite relationship | "Adjacent requires Angle" |
| **formed-by** | Creation relationship | "Corresponding formed-by Angle" |

## How to Use

1. **Click any concept** to see its definition
2. **Use filter buttons** to highlight concept groups
3. **Hover over nodes** to see tooltips
4. **Drag nodes** to rearrange the layout
5. **Use mouse wheel** to zoom in/out
6. **Click background** to reset selection

## Learning Objectives

- **Analyze** hierarchical relationships between angle concepts
- **Evaluate** which concepts are prerequisites for others
- **Understand** how the 14 chapter concepts interconnect

## Bloom's Taxonomy Level

**Analyzing** and **Evaluating** - Understanding concept hierarchies and dependencies.
