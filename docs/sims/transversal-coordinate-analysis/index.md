## About This MicroSim

This interactive tool bridges geometric angle relationships with algebraic coordinate methods. It shows how a transversal intersects two parallel lines and calculates the angle relationships using slope formulas.

## How to Use

1. **Adjust Parallel Slope**: Change the slope of both parallel lines together
2. **Adjust Transversal Slope**: Rotate the transversal independently
3. **Watch Calculations**: See equations, intersections, and angles update in real-time

## Controls

| Control | Function |
|---------|----------|
| **Parallel slope** | Adjust the slope of Lines 1 and 2 (they stay parallel) |
| **Transversal slope** | Adjust the slope of the transversal |
| **Random** | Generate random configurations |

## Analysis Panel Shows

- **Equations** of all three lines in slope-intercept form
- **Parallel Verification** confirming equal slopes
- **Intersection Points** calculated algebraically
- **Angle Measures** computed using slope formulas
- **Angle Relationships** verified (corresponding, alternate interior, same-side interior)
