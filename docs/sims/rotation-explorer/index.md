## About This MicroSim

This interactive tool lets you explore rotations by adjusting the angle and dragging the center of rotation. Watch how the figure rotates while preserving distance from the center.

## How to Use

1. **Drag the red dot** to move the center of rotation
2. **Adjust the angle slider** from -360° to +360°
3. **Click preset buttons** (90°, 180°, 270°, -90°) for common rotations
4. **Click "Animate"** to see smooth rotation
5. **Toggle "Arcs"** and **"Radii"** to show/hide visual aids

## Visual Elements

| Element | Description |
|---------|-------------|
| **Red dot** | Center of rotation (draggable) |
| **Blue triangle** | Pre-image |
| **Green triangle** | Image (rotated) |
| **Purple arc** | Shows rotation angle |
| **Dashed lines** | Radii from center to vertices |

## Key Concepts

- Positive angles rotate counterclockwise
- Negative angles rotate clockwise
- Distance from center is preserved for all points
- The center can be placed anywhere, not just the origin

## Learning Objectives

- **Apply** rotations with different angles and centers
- **Analyze** how center position affects the image location
