## About This Reference Chart

This interactive reference chart summarizes all eight compass and straightedge construction techniques covered in Chapter 4. Hover over any technique to see its step-by-step procedure.

## The Eight Constructions

| # | Technique | Purpose |
|---|-----------|---------|
| **1** | Copy Segment | Create congruent segment |
| **2** | Copy Angle | Create congruent angle |
| **3** | Bisect Segment | Find midpoint (perpendicular bisector) |
| **4** | Bisect Angle | Divide angle in half |
| **5** | Perpendicular Through Point | Perpendicular at point on line |
| **6** | Perpendicular To Line | Perpendicular from external point |
| **7** | Parallel Line | Line through point parallel to given line |
| **8** | Equilateral Triangle | Triangle with equal sides |

## How to Use

1. **Hover over any card** to see the construction steps

## Key Principles

All constructions use only two tools:
- **Compass**: Draw circles and arcs, transfer distances
- **Straightedge**: Draw lines through two points

No measurements allowed - only geometric relationships!

## Learning Objectives

- **Recall** all construction techniques at a glance
- **Identify** the correct technique for a given problem
- **Apply** the appropriate construction method
