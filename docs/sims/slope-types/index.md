## About This Diagram

This four-panel diagram shows the four types of slope with clear visual examples, rise/run arrows, and calculations for each type.

## The Four Types

| Type | Value | Direction | Example |
|------|-------|-----------|---------|
| **Positive** | m > 0 | Rises left to right | m = 1 |
| **Negative** | m < 0 | Falls left to right | m = -1 |
| **Zero** | m = 0 | Horizontal | m = 0 |
| **Undefined** | m = undefined | Vertical | Division by zero |

## Key Concepts

- **Rise**: The vertical change (y₂ - y₁)
- **Run**: The horizontal change (x₂ - x₁)
- **Slope Formula**: m = rise/run = (y₂ - y₁)/(x₂ - x₁)

## Visual Indicators

- **Blue arrows**: Horizontal run
- **Green arrows**: Positive rise
- **Orange arrows**: Negative rise
- **Line colors**: Match panel theme

## Learning Objectives

- **Understand** the meaning of positive, negative, zero, and undefined slopes
- **Analyze** slope type from visual representations
- **Calculate** slope using the rise/run formula
