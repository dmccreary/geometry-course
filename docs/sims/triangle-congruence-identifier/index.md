## About This MicroSim

This practice tool presents triangle pairs with various congruent parts marked. Your task is to identify which congruence criterion (if any) applies.

## How to Use

1. **Examine the triangles** and the marked congruent parts
2. **Click an answer** (SSS, SAS, ASA, AAS, HL, or Not Enough)
3. **Click "Check Answer"** to see if you're correct
4. **Click "Next Problem"** to continue practicing
5. **Click "Hint"** if you need help

## Answer Choices

| Choice | Meaning |
|--------|---------|
| **SSS** | Three sides congruent |
| **SAS** | Two sides + included angle |
| **ASA** | Two angles + included side |
| **AAS** | Two angles + non-included side |
| **HL** | Right triangle: hypotenuse + leg |
| **Not Enough** | Information insufficient (like AAA) |

## Visual Markers

- **Orange tick marks** on sides indicate congruent sides
- **Purple arcs** at angles indicate congruent angles
- **Red square** indicates a right angle

## Learning Objectives

- **Apply** triangle congruence criteria
- **Analyze** which criterion fits given markings
- **Evaluate** whether information is sufficient

## Bloom's Taxonomy Level

**Applying** and **Evaluating** - Identifying correct criteria.
