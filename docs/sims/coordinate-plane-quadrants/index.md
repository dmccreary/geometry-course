## About This Diagram

This interactive diagram introduces the coordinate plane (Cartesian plane) with its key components: x-axis, y-axis, origin, and four quadrants.

## The Four Quadrants

| Quadrant | Location | Signs | Example |
|----------|----------|-------|---------|
| **I** | Upper right | (+, +) | A(3, 4) |
| **II** | Upper left | (−, +) | B(-2, 5) |
| **III** | Lower left | (−, −) | C(-4, -3) |
| **IV** | Lower right | (+, −) | D(5, -2) |

## How to Use

1. **Hover over quadrants** to highlight them
2. **Observe the sample points** in each quadrant
3. **Note the sign patterns** for coordinates in each quadrant

## Learning Objectives

- **Identify** the parts of the coordinate plane
- **Locate** points using ordered pairs
- **Determine** the quadrant based on coordinate signs
