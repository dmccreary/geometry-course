## About This Diagram

This diagram introduces the fundamental terminology for geometric transformations: pre-image (original figure), image (result after transformation), and prime notation (A → A').

## Key Terminology

| Term | Definition |
|------|------------|
| **Pre-Image** | The original figure before transformation |
| **Image** | The resulting figure after transformation |
| **Prime Notation** | A' ("A prime") denotes the image of point A |

## Correspondence

When triangle ABC is transformed:
- A → A' (vertex A maps to A')
- B → B' (vertex B maps to B')
- C → C' (vertex C maps to C')
