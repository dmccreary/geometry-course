## About This MicroSim

This interactive tutorial walks you through the classical compass and straightedge construction for copying a line segment. Each step is animated to show exactly how the construction works.

| **2** | Set compass radius equal to AB |
| **3** | Place compass point on C |

## Why It Works

The compass preserves the distance from A to B. When we place the compass at C with the same radius and draw an arc, point D is exactly the same distance from C as B is from A.

## Learning Objectives

- **Perform** the steps to copy a segment
- **Understand** why the construction creates congruent segments
- **Apply** compass and straightedge techniques
