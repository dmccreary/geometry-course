## About This Infographic

This interactive infographic explores the 2000+ year history of the Parallel Postulate and how questioning it led to revolutionary discoveries in mathematics and physics.

## Timeline Events

| Year | Event |
|------|-------|
| **300 BCE** | Euclid states the Fifth Postulate |
| **1733** | Saccheri explores alternatives |
| **1829** | Lobachevsky publishes hyperbolic geometry |
| **1854** | Riemann develops elliptic geometry |
| **1915** | Einstein uses non-Euclidean geometry |

## Three Types of Geometry

| Geometry | Parallels | Surface | Example |
|----------|-----------|---------|---------|
| **Euclidean** | Exactly 1 | Flat plane | Everyday geometry |
| **Hyperbolic** | Infinitely many | Saddle shape | Special relativity |
| **Elliptic** | Zero (none) | Sphere | Earth navigation |
