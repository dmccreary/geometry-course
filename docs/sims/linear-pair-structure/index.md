## About This MicroSim

This interactive diagram shows the defining characteristics of a **linear pair** - adjacent angles whose non-common sides form a straight line.

## Key Properties

| Property | Description |
|----------|-------------|
| Adjacent | Share vertex and side |
| Straight line | Non-common sides are opposite rays |
| Supplementary | Always sum to 180° |

## How to Use

- **Hover** over each part to see its description
- Notice the angle measurements: 110° + 70° = 180°
- See why linear pairs are always supplementary

## Linear Pair Postulate

> If two angles form a linear pair, then they are supplementary.

$$m\angle 1 + m\angle 2 = 180°$$

## Learning Objectives

- **Recognize** linear pairs
- **Apply** the linear pair postulate
- **Calculate** unknown angles in linear pairs
