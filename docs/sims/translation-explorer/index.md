## About This MicroSim

This interactive tool lets you explore translations by adjusting the horizontal (h) and vertical (k) components using sliders. Watch how the entire figure moves while preserving its size and shape.

## How to Use

1. **Adjust sliders** to set horizontal and vertical translation amounts
2. **Click "Animate"** to see smooth motion from pre-image to image
3. **Toggle "Vector"** to show/hide the translation arrow

## Key Concepts

| Element | Description |
|---------|-------------|
| **Pre-Image** | Original blue triangle (ABC) |
| **Image** | Translated green triangle (A'B'C') |
| **Translation Vector** | Red arrow showing direction and magnitude |
| **Dashed Lines** | Connect corresponding vertices |

## Translation Rules

- **Horizontal (h)**: Positive moves right, negative moves left
- **Vertical (k)**: Positive moves up, negative moves down
- **Coordinate rule**: (x, y) → (x + h, y + k)

## Properties Preserved

- ✓ Size (all distances stay the same)
- ✓ Shape (all angles stay the same)
- ✓ Orientation (clockwise stays clockwise)

## Learning Objectives

- **Apply** translation rules to find image coordinates
- **Analyze** how translations preserve size and shape
