## About This MicroSim

This visualization shows two types of complementary angles - angles whose measures add up to exactly 90°. Hover over each panel to highlight the example.

## Key Concept

**Complementary angles** are two angles whose measures sum to 90°.

$$m\angle A + m\angle B = 90°$$

## Two Types Shown

| Type | Description | Example |
|------|-------------|---------|
| Adjacent | Share vertex and side | 35° + 55° = 90° |
| Non-Adjacent | Separate locations | 25° + 65° = 90° |

## Important Note

Complementary angles do NOT need to be next to each other. They just need to add up to 90°.

## Learning Objectives

- **Identify** complementary angles
- **Calculate** missing angle measures
- **Distinguish** adjacent vs non-adjacent pairs
